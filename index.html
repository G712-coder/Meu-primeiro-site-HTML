Estruturas básicas do HTML

Voltar

Uma estrutra de um hml são os codigos base dele quesão usados para começo dele, os codigos são:

  • HTML

    • Usado para o inicio da programação.

  • Head

    • Usado para manda informação antes de exibir pro usuario

  • Body

    • usado para mostra informações pro usuario.

Esses três codigos são sempre usados em um arquivo HTML.

O que são tags e atributos?

Voltar

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.

Ja os atributos são propriedades que uma Tag pode ter todas as tags tem atributos e algumas tem atibutos mais especifico na tag, que são configurações para aparecer diferente no browser.
